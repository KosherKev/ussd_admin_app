Refined Financial Brutalism

Auto light / dark · Sora + Instrument Serif + DM Mono

Design Methodology
Refined Financial Brutalism

Not the harsh early-2000s brutalism — a refined take rooted in Bloomberg terminals, Figma's 2024 component language, and the crisp utility of trading interfaces. Monochromatic structure with precisely-rationed colour; editorial typography with one serif display face; every UI element earns its place through function.

Typography
Instrument Serif (italic) for all display numbers & headings — financial authority without stuffiness. Sora for UI labels. DM Mono for codes, refs, timestamps, figures.
Colour Economy
Neutral surfaces hold everything. Amber = primary action only. Green = success/healthy. Red = error/alert. Yellow-amber = warning. No decorative colour anywhere else.
Spatial Hierarchy
One hero number per screen. Generous vertical rhythm. Tight mono metadata below every primary value. Tables and lists are dense; action areas breathe.
Texture & Depth
Subtle film-grain noise overlay on the base layer. Four surface levels for card hierarchy (base / surface / raised / high). No blur or glass effects.
Dark Theme
#080B0F base → near-black cool dark surfaces. Amber glows softly. Text at 3 levels: #F0F2F5 / #8B919E / #555D6B.
Light Theme
#F3F4F7 warm-grey base → white surfaces. Amber deepens to #C96A00 for contrast. Subtle borders separate layers without colour.
